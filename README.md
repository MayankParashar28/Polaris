# Polaris

**Polaris** is an AI-powered career assistant that helps you land your dream job.

It analyzes your resume, suggests improvements, and prepares you for interviews using Google's Gemini AI.

## Features
- **Resume Analysis**: Get instant feedback and a readiness score for your target role.
- **Portfolio Generator**: Create a professional portfolio website in one click.
- **Mock Interviews**: Practice specialized technical questions with an AI recruiter.

## Getting Started

1.  **Clone the repo**
    ```bash
    git clone https://github.com/MayankParashar28/Polaris.git
    cd Polaris
    ```

2.  **Install & Run**
    ```bash
    npm install
    npm run dev
    ```
